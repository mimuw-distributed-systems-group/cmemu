# Cycle Debug Log viewer

## How to use

1. Run emulator and produce log file with Cycle Debug Logger.
   For details, use `-h` with the emulator.

2. Open `./build/index.html` and load the log file.

## How to develop

See `./src`. Use `npm start` there for development server
and `npm run build` for making a static build.
Run `npm run build-and-copy` to update the prebuilt version here.
